# WTODO (Weekly Todo)

## デモ
このアプリケーションは GitHub Pages で公開されており、以下のURLから直接試すことができます：
👉 **[WTODO デモサイト](https://mareku.github.io/weekly-todo/)**

WTODOは、1ファイルのHTMLで動作するシンプルかつ高機能な週間タスク管理アプリケーション（SPA）です。
美しいダークテーマと直感的な操作性、キーボードショートカットを備え、ローカル環境で軽快に動作します。

## 主な特徴

- **1ファイル構成**: サーバー不要で、`index.html` をブラウザで開くだけで今すぐ利用可能です。
- **週間タスク管理**: 曜日ごとのタスクを一覧でき、週全体の進捗率（プログレスバー）を視覚的に把握できます。
- **連続達成記録 (Streak)**: タスクをすべて完了した日が続くと、ストリークカウント（🔥 〇日連続）が記録されモチベーションを維持できます。
- **キャリーオーバー機能**: 前週やり残した未完了タスクがある場合、月曜日にワンクリックで一括引き継ぎ（キャリーオーバー）が可能です。
- **繰り返しタスク**: 指定した曜日に自動でタスクを追加する繰り返し設定が可能です。
- **締切タスク (Deadline)**: 期限付きのタスクを管理でき、残り日数や超過日数がバッジで分かりやすく表示されます。また、期限日の週へワンクリックでジャンプできます。
- **期間タスク (Period)**: 開始日と終了日を設定し、進捗バーと残り日数でタスクの進行度を視覚化できます。
- **データ管理**: データはブラウザのローカルストレージ（`wtodo_v4`）に自動保存されます。JSONファイル形式でのエクスポートおよびインポートにも対応しています。

## 使い方

1. `index.html` をブラウザ（Chrome、Edge、Safariなど）で開きます。
2. 画面下部または曜日カードの入力欄からタスクを追加します。
3. タスクを完了したらチェックボックスをオンにします。

### キーボードショートカット

画面上で素早く操作するためのショートカットキーが用意されています。

| キー | アクション |
| :--- | :--- |
| `n` | 今日のカードに新規タスクを追加（フォーカス） |
| `d` | 締切タスクを追加するモーダルを開く |
| `p` | 期間タスクを追加するモーダルを開く |
| `←` / `→` | 前の週 / 次の週へ移動 |
| `?` | ショートカットキーのヘルプ表示のトグル |
| `Escape` | 入力フォーカスを外す / モーダルを閉じる |

※ タスクや締切項目をタップすることで、その場でテキストを編集することができます。

## 技術仕様

- **フロントエンド**: HTML5, CSS3 (Vanilla CSS, ダークモード・レスポンシブ対応), Vanilla JS (ES6+)
- **フォント**: [Google Fonts](https://fonts.google.com/) から `Inter` および `JetBrains Mono` を読み込んで使用しています。
- **外部ライブラリ**: 使用していません（純粋なHTML/CSS/JSのみ）。
- **ストレージ**: ブラウザの `localStorage` (キー名: `wtodo_v4`)
